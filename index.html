What is Monorepo?

Definitions may vary, but the most simple and precise one would be:
a repository that contains multiple projects, that are usually related.

Terms
  • monolith - monolith is not a monorepo, but rather a huge amount of coupled code of 1 application, e.g. your proprietary-application
  • monorepo = monolithic repository - repository with many packages, e.g. Symfony
  • many-repo - approach, when every package is developed in it's own repository, e.g. Doctrine, Zend or Nette
  • single-repo - repository the monorepo is split into, e.g. Symfony\Console
